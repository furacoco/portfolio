Work

01.　森田彦コーヒーWebサイト（架空）

制作業務
Design / Coding
使用言語
HTML / CSS / Javascript（ jQuery ）
目的
Webサイト新規開設による集客効果
コンセプト
ノスタルジックな雰囲気の自家焙煎カフェ
ターゲット
30〜40代の男女

高級店ではないがこだわりのある"プレミアム珈琲"を提供していることから、ターゲットの年齢層にも合わせ、ベースカラーに黒（#303030）を選択し、落ち着いた雰囲気と大人向けの高級感をイメージしました。流行りに流されないお店の信頼感と安心感を感じられるような動きやデザインを工夫しました。

02.　旅行会社京都ツアーLP（架空）

制作業務
Design / Coding
使用言語
HTML / CSS / Javascript（ jQuery ）
目的
LP閲覧によるツアー旅行の資料請求
コンセプト
春の京都で体験型プランがメインのツアー
ターゲット
20〜40代前後の女性・職場の友人など

ファーストビューで桜が舞う春の京都で舞妓体験を楽しむ旅行者のイメージが伝わるように意識しました。"はんなり旅"というテーマを設定し、華やかなグラデーションとアクセントカラーに金色をイメージするカラー（#c7a946）を使用しました。モーダルウィンドウによりページ遷移することなく資料請求フォームを入力出来るデザインにいたしました。
